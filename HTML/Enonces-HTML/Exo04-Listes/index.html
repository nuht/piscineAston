Faire des listes

Suite ordonnée standard (type='1')

Exemple :
  1. Elément premier
  2. Elément second
  3. Elément tierce

Suite ordonnée i, ii, iii (type='i')

Exemple :
  1. Elément premier
  2. Elément second
  3. Elément tierce

Suite ordonnée A, B, C (type='A')

Exemple :
  1. Elément premier
  2. Elément second
  3. Elément tierce

Suite non-ordonnée standard (type='disc')

Exemple :
  • Elément premier
  • Elément second
  • Elément tierce

Suite non-ordonnée avec des carrés (type='square')

Exemple :
  • Elément premier
  • Elément second
  • Elément tierce

Suite non-ordonnée avec des ronds (type='circle')

Exemple :
  • Elément premier
  • Elément second
  • Elément tierce
